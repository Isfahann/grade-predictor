\# 🎓 Student Grade Predictor



A machine learning web app that predicts a student's exam grade based on their study habits and academic history.



\## Features

\- Predicts exam score out of 100 using a Random Forest model

\- Takes inputs like study hours, attendance, previous scores, and sleep hours

\- Displays grade letter (A/B/C/D) with personalised suggestions

\- Shows model performance metrics (R² Score, MAE)



\## Tech Stack

\- Python

\- scikit-learn

\- Streamlit

\- Pandas \& NumPy



\## How to Run

1\. Clone the repo git clone https://github.com/Isfahann/grade-predictor.git

2\. Install dependencies

pip install -r requirements.txt

3\. Run the app

streamlit run app.py



\## Model Details

\- Algorithm: Random Forest Regressor

\- Dataset: Synthetically generated (500 samples)

\- Features: Study hours, attendance, previous score, assignment completion, sleep hours
